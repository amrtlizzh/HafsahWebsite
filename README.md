# Mahallah Hafsah Website
A dedicated web platform designed for the residents of Mahallah Hafsah (IIUM). This project aims to centralize information, improve communication, and enhance community engagement for students living within the mahallah.

Project Objectives
1. Centralize Information: Provide easy access to management details, facilities, and rules.
2. Improve Communication: Streamline maintenance reporting and event updates.
3. Community Focused: Create a digital space that fosters a pleasant living environment.

Features
- Management Directory: Interactive section displaying the organizational chart, including the Principal, Fellows, and Mahallah Representative Committee (MRC).
- Interactive Side Panel: A custom-coded feature to display details about facilities (CafÃ©, Mart, etc.) and upcoming events without leaving the main page.
- Maintenance Reporting: Integrated QR codes and links for students to submit maintenance requests directly.
- Islamic Integration: The homepage features Hadiths and an Islamic-inspired logo reflecting the IIUM identity.
- Responsive Gallery: A visual showcase of the mahallah's environment using the Fotorama library.
- Location Navigation: Embedded Google Maps to help residents and visitors navigate the mahallah area.

Technologies Used
Frontend: HTML5, CSS3, JavaScript
Libraries & Resources:
JQuery & AJAX: For smooth navigation and dynamic content switching (e.g., toggling between management profiles).
Fotorama: For high-quality, responsive image sliders.
Font Awesome: For professional iconography.
Google Maps API: For location services.
